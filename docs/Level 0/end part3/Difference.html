The difference between Programming Language, Framework, Library, Module, Package

مفهوم کنترل در دست کیست؟ انعطاف‌پذیری هدف اصلی
زبان برنامه‌نویسی (Python, Java, ...) توسعه‌دهنده بسیار بالا پایه و اساس کدنویسی
فریمورک (Django, React, Spring) فریمورک کم (پیروی از ساختار) ارائه یک چهارچوب کامل
کتابخانه (NumPy, jQuery) توسعه‌دهنده بالا ارائه توابع آماده
ماژول (math.py در پایتون) توسعه‌دهنده متوسط سازماندهی و تقسیم کد
پکیج (npm, pip) توسعه‌دهنده متوسط توزیع و مدیریت ماژول‌ها
