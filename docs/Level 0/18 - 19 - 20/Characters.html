کاراکترهای خاص در برنامه‌نویسی

کاراکتر نام انگلیسی نام فارسی مثال برنامه‌نویسی
! Exclamation mark علامت تعجب if (x != 0) { … }
@ At sign علامت اَت @decorator
# Hash / Pound علامت هَش/دیز # This Is a comment
$ Dollar sign علامت دلار let price = 100$;
% Percent sign علامت درصد/مودولو 10 % 3 → 1
^ Caret علامت کلاهک 2^3 (توان در برخی زبان‌ها)
& Ampersand علامت اَند if (x & y) { … }
* Asterisk علامت ستاره def func(*args)
( ) Parentheses پرانتز print("Hello")
- Hyphen خط تیره 5 – 3
_ Underscore آندرلاین user_name = "Ali"
= Equals sign علامت مساوی x = 10
+ Plus علامت جمع 2 + 3
[ ] Square brackets کروشه list = [1, 2, 3]
{ } Curly braces آکولاد dict = {"name": "Ali"}
| Pipe خط عمودی if (x | y) { … }
\ Backslash بک‌اسلش "C:\\folder\\file.txt"
/ Slash اسلش 10 / 2
: Colon دونقطه if x > 0:
; Semicolon نقطه‌ویرگول let x = 10;
" ' Double/Single quote دابل/تک کوتیشن name = "Ali"
, Comma ویرگول [1, 2, 3]
. Dot نقطه math.pi
~ Tilde تیلدا ~x (NOT بیتی)
` Backtick بک‌تیک `Hello ${name}`
