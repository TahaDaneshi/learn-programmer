Freelance Sites

    🌍 International freelance sites:

  1. Upwork

    • بزرگترین پلتفرم فریلنسری جهانی

    • مناسب برای تمام مهارت‌ها (برنامه‌نویسی، طراحی، مارکتینگ و ...)

  2. Fiverr

    • خرید و فروش خدمات (Gig-Based)

    • مناسب برای پروژه‌های کوچک و سریع

  3. Freelancer

    • رقابتی و مناسب برای پروژه‌های فریلنسری و مسابقات

  4. Toptal

    • مخصوص فریلنسرهای حرفه‌ای و نخبه

    • فرآیند پذیرش سخت‌گیرانه

  5. Guru

    • مناسب برای فریلنسرهای با تجربه

    -IR- Iranian freelance sites

  2. Parscoders

    • تخصصی برنامه‌نویسی و توسعه نرم‌افزار

    • مناسب برای فریلنسرهای ایرانی

  3. Jobinja Freelance

    • بخش فریلنسری سایت جابینجا

    • پروژه‌های کوتاه‌مدت و بلندمدت

  4. Faranesh

    • مناسب برای تولید محتوا، ترجمه و طراحی

  5. IranFreelancer

    • پروژه‌های فریلنسری در حوزه‌های مختلف

  6. jobvision

    • دارا بودن نرم افزار اندروییدی

    • ترکیبی از کار ریموت و حضوری
