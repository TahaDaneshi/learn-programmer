HTML5 Semantic Elements

HTML5 introduced semantic elements that clearly describe their meaning to browsers and developers.

Why use semantic HTML?

Semantic HTML improves accessibility, SEO, and code maintainability.

<article>
    <header>
        <h1>Article Title</h1>
    </header>
    <section>
        <p>Article content...</p>
    </section>
    <footer>
        <p>Author info</p>
    </footer>
</article>

React Components

React components let you split the UI into independent, reusable pieces.

Components can be either class-based or functional with hooks.

Functional Component Example 
function Welcome(props) {
    return <h1>Hello, {props.name}</h1>;
}
